# Outreach Log Template

Use one block per outreach touch and paste the filled block into `docs/operations/OUTREACH-TRACKER.md` or a dated note if the tracker gets too crowded.

## Template

```md
### <YYYY-MM-DD> — <ecosystem> — <target or handle>
- Status: SENT | RESPONDED | FOLLOW_UP | CLOSED
- Channel: <discord/github/forum/email/x>
- Route used: <primary route or backup route>
- Message asset: <content/outreach-send-ready-2026-03-06.md section>
- Ask summary: <one sentence>
- Links sent:
  - <link 1>
  - <link 2>
- Response summary: <leave blank until response>
- Thesis changed?: yes | no
- Follow-up date: <optional>
- Notes: <optional>
```

### 2026-03-07 — Virtuals / ACP — builders-chat
